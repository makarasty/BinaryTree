# BinaryTree
Написати консольний додаток який буде генерувати випадковий масив чисел довжиною яку вказує користувач з консолі.
Реалізувати алгоритму перетворення довільного масиву цілих чисел у впорядковане бінарне дерево.
Реалізувати алгоритм, який буде впорядковувати довільний масив за допомогою бінарного дерева за зростанням чи спаданням значень.
Реалізувати алгоритм пошуку значення в бінарному дереві.
